# 💱 Currency Converter - Vanilla JavaScript Milestone Project

A clean, responsive, and lightweight Currency Converter web application built using fundamental front-end web technologies. This project interacts with a live exchange rate API to fetch real-time conversion metrics, providing an immediate and smooth user calculation loop.

---

## 🚀 Core Features

- **Live Exchange Rates:** Fetches up-to-date conversion multi-currency indices dynamically using a free exchange rate API endpoint.
- **Bi-directional Mapping:** Allows seamless selection of origin ("From") and target ("To") currency parameters from interactive dropdown structures.
- **Dynamic DOM UI Updates:** Calculations happen instantly on form submission with instant screen text rendering—zero page refreshes required.
- **Responsive Layout:** Styled with vanilla CSS grids/flexbox alignment, making it fully optimized across mobile, tablet, and desktop screens.

---

## 🛠️ Built With

- **HTML5:** Structured semantic markup for form wrappers, numerical inputs, and currency selection indicators.
- **CSS3:** Custom layout design containing modern input field alignments, button active transformations, hover states, and smooth element transitions.
- **Vanilla JavaScript (ES6+):** 
  - Async/Await workflow handling asynchronous network request parameters (`fetch` API).
  - Explicit Event Listener attachments tracking user interaction points.
  - Runtime DOM Manipulation logic modifying text elements on the fly.
